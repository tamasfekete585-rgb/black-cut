# Black Cut weboldal prototípus

Ez egy statikus, megnyitható weboldal-prototípus gravírozott termékekhez.

## Fájlok

- `index.html` - az oldal szerkezete
- `styles.css` - japán shoji ajtó, bambusz hangulat, reszponzív megjelenés
- `script.js` - kategóriaszűrés, terméklista, rendelési ablak, képfeltöltés neve

## Megnyitás

Nyissa meg az `index.html` fájlt böngészőben. Nincs szükség telepítésre vagy szerverre.

## Későbbi élesítéshez

- valódi termékfotók és árak beírása
- kosár és fizetési folyamat bekötése
- rendelési e-mail vagy admin felület
- tárhely és domain beállítása
